本项目基于[readest](https://github.com/readest/readest) 进行开发，主要是为了拓展AI阅读功能，进行集成，学习实践RAG相关的功能

**Readest** is an open-source ebook reader designed for immersive and deep reading experiences. Built as a modern rewrite of [Foliate](https://github.com/johnfactotum/foliate), it leverages [Next.js 15](https://github.com/vercel/next.js) and [Tauri v2](https://github.com/tauri-apps/tauri) to deliver a smooth, cross-platform experience across macOS, Windows, Linux, Android, iOS, and the Web.
